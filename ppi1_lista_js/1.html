O que é JavaScript?

O JavaScript foi criado em 1995 por Brendan Eich para tornar as páginas da Web interativas. Antes dele, as páginas HTML eram estáticas, sem recursos para responder às ações do usuário, como cliques, validação de formulários e animações.

O JavaScript pode ser executado tanto no navegador quanto no servidor. No navegador, ele é interpretado pelo motor JavaScript do browser e pode manipular a página HTML, o CSS e interagir com o usuário. No servidor, utilizando plataformas como Node.js, ele executa tarefas como acesso a banco de dados, criação de APIs e processamento de informações.

No navegador, o JavaScript é executado em um ambiente chamado sandbox. Isso significa que o código possui restrições de segurança, não podendo acessar livremente arquivos do computador ou o sistema operacional do usuário sem autorização.

Alguns dos principais objetos globais do JavaScript são:
document: permite acessar e modificar os elementos da página.
console: exibe mensagens no console do navegador para depuração.
alert: mostra uma caixa de diálogo com uma mensagem ao usuário.
