Introdução

A Internet surgiu a partir da necessidade de conectar computadores para troca de informações. Com o passar dos anos, a criação da World Wide Web (WWW) tornou o acesso aos conteúdos muito mais simples, utilizando páginas escritas em HTML e acessadas por navegadores.

Atualmente utilizamos aplicações Web modernas, que oferecem interação em tempo real, armazenamento em nuvem e diversos serviços online.

Linha do Tempo

  1. Década de 1960: Surgimento das primeiras redes de computadores.
  2. 1969: Criação da ARPANET, considerada a precursora da Internet.
  3. 1983: Adoção oficial do protocolo TCP/IP.
  4. 1984: Criação do DNS (Sistema de Nomes de Domínio).
  5. 1989: Tim Berners-Lee propõe a criação da World Wide Web no CERN.
  6. 1990: Desenvolvimento do HTML, do HTTP e do primeiro servidor Web.
  7. 1991: A Web torna-se disponível ao público.
  8. 1993: Surgimento do navegador Mosaic, que popularizou a Web.
  9. Anos 2000: Evolução da Web 1.0 para aplicações Web modernas (Web 2.0).

Conceitos Importantes

Internet
Rede mundial que conecta computadores e dispositivos.
Web
Conjunto de páginas acessadas pela Internet através de navegadores.
Servidor Web
Computador responsável por armazenar e disponibilizar páginas da Web.
Navegador
Programa utilizado para acessar páginas da Internet.
HTML
Linguagem de marcação utilizada para criar páginas Web.
HTTP
Protocolo responsável pela comunicação entre navegador e servidor.

Web 1.0 x Aplicações Web Modernas

  • Web 1.0: páginas estáticas, pouca interação com o usuário.
  • Aplicações Web modernas: páginas dinâmicas, interação em tempo real, redes sociais, streaming e armazenamento em nuvem.
