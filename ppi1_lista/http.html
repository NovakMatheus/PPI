O que é o HTTP?

HTTP (HyperText Transfer Protocol) é o protocolo utilizado para a comunicação entre o navegador do usuário e o servidor onde um site está hospedado. Sempre que uma página é acessada, o navegador envia uma solicitação ao servidor, que responde com os arquivos necessários para exibir o conteúdo.

Como funciona a comunicação?

Cliente

O cliente é o navegador (Google Chrome, Firefox, Edge, etc.), responsável por solicitar informações ao servidor.

Servidor

O servidor é o computador que armazena o site e responde às solicitações enviadas pelos clientes.

Requisição (Request)

É a mensagem enviada pelo cliente solicitando um recurso, como uma página, imagem ou arquivo.

Resposta (Response)

É a mensagem enviada pelo servidor contendo o recurso solicitado ou um código informando o resultado da solicitação.

Métodos HTTP

GET

O método GET é utilizado para solicitar informações ao servidor. É o método mais comum ao acessar páginas da Web.

POST

O método POST é utilizado para enviar dados ao servidor, como formulários de cadastro, login ou envio de mensagens.

Exemplo de Requisição HTTP

GET /index.html HTTP/1.1
Host: www.exemplo.com
User-Agent: Mozilla/5.0
Accept: text/html

Exemplo de Resposta HTTP

HTTP/1.1 200 OK
Content-Type: text/html

<html>
    <body>
        Página carregada com sucesso.
    </body>
</html>

Principais Códigos de Status HTTP

Código Significado
200 Requisição realizada com sucesso.
301 Recurso movido permanentemente para outro endereço.
400 Requisição inválida enviada pelo cliente.
403 Acesso proibido ao recurso solicitado.
404 Página ou recurso não encontrado.
500 Erro interno do servidor.

Resumo

  • HTTP é o protocolo de comunicação da Web.
  • O navegador funciona como cliente.
  • O servidor responde às solicitações.
  • GET solicita informações.
  • POST envia dados ao servidor.
  • Os códigos de status informam o resultado da requisição.
