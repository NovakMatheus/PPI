Este site foi desenvolvido como parte de um trabalho de HTML. Seu objetivo é apresentar conteúdos sobre a Web, o protocolo HTTP, estados brasileiros, além de informações pessoais, currículo e contato.

Internet x Web

A Internet é a rede mundial que conecta computadores e dispositivos. A Web é um dos serviços que utiliza essa rede para acessar páginas por meio de navegadores. Ou seja, a Web depende da Internet para funcionar, mas a Internet oferece diversos outros serviços além da Web.
